# Тренажер алгоритмів сортування

Інтерактивний вебзастосунок для покрокового вивчення алгоритмів сортування. Розроблено для студентів ПНПУ.

## Можливості

- **Покроковий тренажер** — відповідайте на запитання на кожному кроці алгоритму та отримуйте миттєвий зворотній зв'язок.
- **Режим демонстрації** — спостерігайте за автоматичним виконанням алгоритму з поясненнями.
- **Відображення коду** — підсвічений псевдокод (C++) оновлюється в реальному часі відповідно до поточного кроку.
- **Складність алгоритму** — переглядайте часову та просторову складність для кожного алгоритму.
- **Гнучке введення даних** — вводьте масив вручну або генеруйте його випадково.
- **Адаптивний дизайн** — коректно відображається на мобільних пристроях і десктопі.

## Підтримувані алгоритми

| Алгоритм | Часова складність | Просторова складність |
|---|---|---|
| Сортування вставками (Insertion Sort) | O(N²) | O(1) |
| Сортування вибором (Selection Sort) | O(N²) | O(1) |
| Сортування бульбашкою (Bubble Sort) | O(N²) | O(1) |
| Сортування Шелла (Shell Sort) | O(N log N) | O(1) |
| Швидке сортування, схема Хоара (Quick Sort) | O(N log N) | O(log N) |

## Запуск

Застосунок не потребує встановлення залежностей або складання. Достатньо відкрити файл `index.html` у будь-якому сучасному браузері:

```bash
# Клонуйте репозиторій
git clone https://github.com/Rethinger/sort-trainer-for-pnpu.git
cd sort-trainer-for-pnpu

# Відкрийте у браузері
open index.html        # macOS
xdg-open index.html    # Linux
start index.html       # Windows
```

Або просто перетягніть файл `index.html` у вікно браузера.

## Як користуватися

1. **Оберіть алгоритм** зі списку.
2. **Вкажіть спосіб введення масиву**: вручну (список чисел через кому) або випадково (зазначте кількість елементів від 3 до 10).
3. Натисніть **«Почати тренування»** для режиму запитань або **«Демонстрація алгоритму»** для автоматичного перегляду.
4. У режимі тренування відповідайте на запитання: введіть число або оберіть «Так» / «Ні».
5. Після завершення переглядайте свою статистику та проходьте симуляцію знову.
